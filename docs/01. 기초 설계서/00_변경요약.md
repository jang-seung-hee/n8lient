## 반영한 핵심 항목

```text
1. result.summary / result.resultUrl = 가벼운 사용자용 결과 확인 메타
2. processorResult.summary = Level 2 이상 상세 사용자용 요약
3. processorResult.structuredData.actionLinks = 후속 액션 링크 목록
4. [24A] Build User Result Message 노드 권장
5. AI 초안 작성 → 사용자 승인 → callback payload 반영 절차
6. token/secret/credential/API key/authorization 노출 금지
```
