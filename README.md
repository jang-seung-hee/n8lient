# JANG'S 아이디어 캐처 (클라이언트 프로그램)

이 프로젝트는 n8n 자동화 워크플로우를 활용하여 아이디어를 효과적으로 정제하고 분석하여 QNA 라이브러리 및 데이터베이스에 구축하는 전용 클라이언트 플랫폼입니다.

---

## 🛠️ 기술 스택

프로젝트에 사용된 핵심 프레임워크와 라이브러리 목록입니다.

* **Core**: Next.js (App Router, v16), React v19, TypeScript v5
* **Styling**: Tailwind CSS v4, Vanilla CSS
* **Development Tools**: ESLint, PostCSS

---

## 🚀 실행 명령어

로컬 개발 및 빌드를 위해 아래 명령어들을 실행합니다.

```bash
# 의존성 패키지 설치
npm install

# 로컬 개발 서버 실행
npm run dev

# 프로덕션 빌드 파일 생성
npm run build

# 빌드된 프로덕션 서버 실행
npm run start

# ESLint 코드 스타일 검사
npm run lint
```
